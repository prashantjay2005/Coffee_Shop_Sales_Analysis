# Coffee_Shop_Sales_Analysis
Coffee Shop Sales Analysis
Project Overview
This project performs Exploratory Data Analysis (EDA) on a Coffee Shop Sales dataset using Python, Pandas, Matplotlib, and Seaborn. The objective is to analyze sales performance, customer purchasing behavior, product categories, and store-wise trends to generate meaningful business insights through statistical analysis and data visualization.
Objectives
•
Analyze coffee shop sales data.
•
Identify high-performing store locations and product categories.
•
Study weekday-wise sales patterns.
•
Examine transaction quantity and pricing trends.
•
Generate business insights using visualizations.

Analysis Performed
The following analyses were carried out:
•
Generated descriptive statistics using df.describe().
•
Performed correlation analysis with df.corr().
•
Calculated total sales by store location using groupby().
•
Created pivot tables to compare product categories and weekday sales across stores.
•
Conducted interactive analysis by selecting store location and product category.
•
Generated business insights using statistical summaries and aggregation techniques.
