## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Coffee Shop Sales dataset using **Python, Pandas, Matplotlib, and Seaborn**. The objective is to analyze sales performance, customer purchasing behavior, product categories, and store-wise trends to generate meaningful business insights through statistical analysis and visualization.

---

## 🎯 Objectives

- Analyze coffee shop sales data.
- Identify high-performing store locations.
- Analyze product category performance.
- Study weekday-wise sales trends.
- Examine transaction quantity and pricing behavior.
- Generate business insights using data visualization.

## 📊 Analysis Performed

The following analyses were conducted:

- Descriptive Statistics (`df.describe()`)
- Correlation Analysis (`df.corr()`)
- Store-wise Sales Analysis using `groupby()`
- Product Category Analysis
- Weekday-wise Sales Analysis
- Pivot Table Analysis
- Interactive Store & Product Analysis
- Business Insight Generation

## 🔍 Key Insights

- Store performance varies significantly across locations.
- Certain product categories contribute more to total revenue.
- Customer purchasing behavior changes throughout the week.
- Transaction quantity and unit price directly influence total sales.
- Data visualization helps identify patterns and supports business decision-making.
